# MemeMatch – социальная сеть, основанная на мемах

## Описание

MemeMatch – соцсеть со встроенным редактором мемов, позволяющим загрузить картинку и уже через несколько минут выложить её. Загруженные мемы можно изменять прямо на сайте – такое редактирование чужого мема называется `Match (Мэтч)`. Также возможно оставлять реакции под чужими мемами, а с `МэтчМэнами` (`MatchMan` – тот, кто сделал мэтч с вашим мемом) можно переписываться.

## Функционал

-   `MemeFeed (Мемолента)` – лента мемов
-   `MemeMaker (Мемодел)` – редактор собственных мемов
-   `MemeAPI (МемАПИ)` – собственный API

## MemeAPI

МемАПИ – инструмент для разработчиков, позволяющий разработчикам отправлять картинки с нужными им параметрами и получать готовый мем, или получать уже созданный мем. Документация по МемАПИ находится в файле [MemeAPI.md](MemeAPI.md).
